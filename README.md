# Clock

The purpose of this program is to create a 12 hour and 24 hour clock. The user can add the clock by a second, minute or hour, and both clocks would update depending on the user input.

In this program, what went well was achieving a solution on how to get user input through the use of a string and then concerting it to an integer in order to update the time clock. I was also able to make sure that the clock did not display a time that did not exist.

To enhance this code, I am aware there is a logic error which I believe has to do with the first hour in each clock when they complete a cycle. I would need to make a fix to this in order for the program to work as intended.

The pieces of code I found most difficult to program were most likely the parts were the hour and minute needed to update if the second was at 59 and went up by one. I believe this also applies to the minute as well. It was frusturating, but I got the job done.

If there were any skills I learned from this project, they were most likely how to convert a string to an integer. I am sure that can come in handy in the future.

I made this program maintainable, readable and adaptable by ensuring each line of code had a comment attached to it. This way, someone can hopefully understand what a certain piece of code does.
